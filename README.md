# webAppOS
webAppOS files at github

## Installation

```bash
git clone https://github.com/LUMII-Syslab/webAppOS.git
cd webAppOS/src
./gradlew build
./gradlew install
cd ../dist/apps/Login.webservice/
cp webservice.properties.template webservice.properties
```
